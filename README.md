# Urbano Santiago | Gestión Inmobiliaria Inteligente

Este proyecto representa una solución de vanguardia para el sector inmobiliario, combinando una interfaz de usuario de alta gama con la potencia de la Inteligencia Artificial generativa de Google (**Gemini 1.5 Flash**).

## 🚀 Propuesta de Valor
Urbano Santiago no es solo un catálogo de propiedades; es un **Talking Web Site**. Permite a los potenciales clientes interactuar mediante la voz para buscar departamentos en Las Condes, Providencia o casas en Chicureo y Lo Barnechea, obteniendo respuestas inmediatas y profesionales.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, Tailwind CSS (Diseño responsivo y moderno).
* **Inteligencia Artificial:** Google Gemini SDK (Modelo 1.5 Flash).
* **Voz:** Web Speech API (Reconocimiento y síntesis de voz en español chileno).
* **Despliegue:** Vercel con integración continua desde GitHub.
* **Seguridad:** Gestión de variables de entorno mediante `vercel.json`.

## 📂 Estructura del Proyecto
* `index.html`: Núcleo del sitio con la lógica de IA y diseño de secciones.
* `vercel.json`: Configuración de seguridad para la API Key.
* `/destacados*.jpg`: Activos visuales de alta resolución para la exhibición de propiedades.
